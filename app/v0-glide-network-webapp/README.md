# v0-glide-network-webapp

## Overview
This project is a web application designed for service providers to register and manage their services. It utilizes Supabase for authentication and database management.

## Features
- Service provider registration
- User authentication
- Database integration with Supabase

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd v0-glide-network-webapp
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Set up your Supabase project and configure the `supabaseClient.ts` file with your Supabase URL and public API key.

## Usage
- Start the development server:
  ```
  npm run dev
  ```
- Navigate to `http://localhost:3000` to access the application.
